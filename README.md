# Ominous Launcher

Ominous Launcher is a modern Minecraft launcher built with Tauri (Rust) and Next.js, focused on performance, simplicity, and seamless mod integration.

## Features

* CurseForge mod and modpack integration
* One-click mod installation and updates
* Lightweight and fast UI
* Automatic launcher updates
* Version and instance management

## Status

Currently in active development.

## Notes

This project uses the official CurseForge API and does not redistribute any content. All downloads are performed directly from CurseForge sources in compliance with their terms.

## Tech Stack

* Tauri (Rust)
* Next.js
* TypeScript
